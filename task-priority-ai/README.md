# AI タスク優先度管理

アイゼンハワーマトリクス（緊急度×重要度）を使って、AIが自動でタスクを優先順位付けしてくれるWebアプリケーションです。

## 概要

日々のTODOタスクを入力すると、キーワードベースの分類ロジックで自動的にタスクを以下の4つの象限に分類します：

1. **緊急かつ重要** - 今すぐやる
2. **重要だが緊急でない** - 計画して実行
3. **緊急だが重要でない** - 委任を検討
4. **緊急でも重要でもない** - 後回しまたは削除

## 主な機能

- テキスト入力によるタスク追加
- キーワードベースの自動優先度分類（モック実装）
- アイゼンハワーマトリクス（2×2）形式でのタスク表示
- ローカルストレージによるタスクの永続化
- タスクの削除機能

## 分類ロジック

このモック実装では、以下のキーワードを使ってタスクを分類します：

- **緊急性キーワード**: 今日、明日、締切、期限、急ぎ、すぐ、至急、緊急、今すぐ、asap
- **重要性キーワード**: 重要、必須、クリティカル、プロジェクト、報告、会議、発表、提出、リリース、本番

これらのキーワードの組み合わせで4つの象限に自動分類されます。

## 技術スタック

- **フロントエンド**: React + TypeScript
- **ビルドツール**: Vite
- **スタイリング**: Tailwind CSS
- **分類ロジック**: キーワードベース（モック実装）

## セットアップ

### 必要な環境

- Node.js 18以上

### インストール手順

1. リポジトリのクローン

```bash
git clone <repository-url>
cd task-priority-ai
```

2. 依存パッケージのインストール

```bash
npm install
```

3. 開発サーバーの起動

```bash
npm run dev
```

4. ブラウザで `http://localhost:5173` を開く

## 使い方

1. **タスクの追加**
   - タスク名（必須）と詳細（任意）を入力
   - 「タスクを追加」ボタンをクリック
   - キーワードに基づいて自動的に優先度を分類してマトリクスに配置

2. **タスクの管理**
   - 各タスクの右上の×ボタンでタスクを削除
   - 「すべてのタスクを削除」で全タスククリア

3. **分類のコツ**
   - 緊急性を示すキーワード（今日、明日、締切など）を含める
   - 重要性を示すキーワード（重要、プロジェクト、会議など）を含める
   - 例：「今日締切のプロジェクト報告書作成」→「緊急かつ重要」に分類

## 注意事項

このアプリケーションは**モック実装**です。実際のAI APIは使用せず、キーワードベースの簡易的な分類ロジックで動作します。

**将来的な拡張案：**
- 実際のLLM API（Claude、GPT-4など）との統合
- バックエンドサーバーでAPIキーを安全に管理
- より高度な自然言語処理による分類精度の向上
